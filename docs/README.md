# Documentation Portail 2 Vantaux

Documentation technique pour la mise en service et l'utilisation du système de motorisation de portail à 2 vantaux.

> Utilisez le menu latéral pour naviguer entre les sections.

## Séquence de mise en service

```
Appairage télécommande → Diagnostic moteurs → Apprentissage butées → Mode normal
```

## Référence rapide boutons carte

| Bouton | Fonction |
|--------|----------|
| Bouton 1 | Télécommandes (appairage, suppression) |
| Bouton 2 | Mode diagnostic (moteurs, sorties, entrées) |
| Bouton 3 | Mode apprentissage (Maître, Dormant) |

## Référence rapide LED

| Couleur LED | Fréquence | Signification |
|-------------|-----------|---------------|
| Bleu clignotant | Rapide | Mode appairage télécommande |
| Violet clignotant | Rapide | Mode suppression télécommande |
| Magenta clignotant | Rapide | Suppression toutes télécommandes |
| Blanc clignotant | Lent | Mode vérification moteurs |
| Cyan clignotant | Lent | Mode diagnostic sorties |
| Jaune clignotant | Lent | Mode diagnostic entrées |
| Rouge clignotant | Rapide | Mode apprentissage Maître |
| Vert clignotant | Rapide | Mode apprentissage Dormant |
